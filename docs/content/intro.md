# Life Simulator Documentation

This site is generated directly from source code under `src/` on each CI run.

- Architecture is generated from the current folder/module layout.
- Feature-area indexes are generated for `TenantSystem`, `PlotSystem`, `Network`, and `Utilities`.
- API reference is generated from Luau module signatures, exported types, and public methods.

Use the sidebar to navigate generated pages.
